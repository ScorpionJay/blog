## 简介
	编写代码，页面自动刷新
	js压缩
	js合并
	html压缩
	css压缩
	image压缩
	复制删除
	打包build
	js lib grunt b

## 如何使用
	1.安装node
	
	2.安装grunt npm install grunt-cli grunt --g
	
	3.到mars文件夹下 安装需要的插件 npm intall
	
	grunt server 启动服务
